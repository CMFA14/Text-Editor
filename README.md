# 📝 DocFlex — Gestão de Documentos em Tempo Real

> [!IMPORTANT]
> **Status do Projeto**: 🚀 **EM PRODUÇÃO**
> O sistema está operacional e em fase ativa de evolução de funcionalidades.

O **DocFlex** é um editor de texto rico e moderno desenvolvido com **React** e **Tiptap**, projetado para oferecer uma experiência de escrita fluida e profissional diretamente no navegador. Com um sistema robusto de gerenciamento de múltiplos documentos, ele permite que você crie, organize e exporte seus textos com uma estética premium e minimalista.

## ✨ Principais Funcionalidades

### 📂 Gestão de Documentos
- **Dashboard Intuitivo**: Tela inicial com grade de documentos recentes e data de modificação.
- **Multidocumentos**: Crie e alterne entre diversos arquivos sem perder o progresso.
- **Auto-salvamento**: Suas alterações são salvas automaticamente no `localStorage`.
- **Importação Flexível**: Importe arquivos `.txt`, `.md` e `.html` diretamente para o editor.

### ✍️ Edição Poderosa
- **Editor Rico (Rich Text)**: Suporte a Negrito, Itálico, Sublinhado, Tachado, Tabelas, Imagens e muito mais.
- **Tipografia Personalizada**: Escolha entre diversas fontes profissionais e ajuste o tamanho do texto.
- **Cores e Destaques**: Paleta de cores moderna para texto e fundo.
- **Listas de Tarefas**: Crie checklists interativos de forma nativa.

### 🛠️ Utilidades Avançadas
- **Exportação Multi-formato**: Baixe seus documentos em HTML formatado, Markdown ou Texto Simples.
- **Localizar e Substituir**: Ferramenta integrada de busca rápida no texto.
- **Modo Noturno (Dark Mode)**: Interface totalmente adaptável com design *glassmorphism*.
- **Contagem de Palavras**: Estatísticas em tempo real na barra de status.

## 🚀 Tecnologias Utilizadas

- **React 18** + **Vite**
- **TypeScript**
- **Tiptap** (Editor Framework)
- **Tailwind CSS** (Utilidades de layout)
- **Vanilla CSS** (Design System e Glassmorphism)
- **Lucide Icons** (SVGs consistentes)

## 💻 Como Rodar Localmente

1. Clone o repositório:
   ```bash
   git clone https://github.com/CMFA14/Text-Editor.git
   ```

2. Instale as dependências:
   ```bash
   npm install
   ```

3. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```

4. Acesse em seu navegador: `http://localhost:5173`
